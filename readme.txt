== Requirements ==
*	Modern web browser such as Google Chrome, Firefox and Safari.

== Installation ==
*	Upload `smsify` folder to the `/wp-content/plugins/` directory
*	Activate the plugin through the 'Plugins' menu in WordPress
*	Get SMS credits at [cloudinn.io](https://cloudinn.io/products/smsify "SMSify - SMS WordPress plugin")
*	Enjoy!
